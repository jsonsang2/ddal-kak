딸깍! 행운의 로또 번호 생성기

버튼 한 번이면 오늘의 행운 번호가 나옵니다. 매주 새로운 번호로 행운을 시험해 보세요.

최근 생성 기록

    로또 6/45란?

    대한민국 로또 6/45는 1부터 45까지의 숫자 중 6개를 선택하여 당첨 번호와 일치하는 개수에 따라 상금이 결정되는 복권입니다. 매주 토요일 오후 8시 45분에 추첨이 진행되며, 1등 당첨 확률은 약 814만분의 1입니다.

    당첨 등수별 조건

    등수조건당첨 확률
    1등6개 번호 모두 일치1/8,145,060
    2등5개 일치 + 보너스 번호1/1,357,510
    3등5개 번호 일치1/35,724
    4등4개 번호 일치1/733
    5등3개 번호 일치1/45

    번호 생성기 사용법

    1

    번호 생성 버튼 클릭

    상단의 "번호 생성하기" 버튼을 누르면 1~45 중 랜덤한 6개 번호가 자동으로 선택됩니다.

    2

    번호 확인

    생성된 번호는 오름차순으로 정렬되어 표시됩니다. 마음에 들지 않으면 다시 생성할 수 있습니다.

    3

    번호 기록 확인

    생성한 번호는 최근 5개까지 기록됩니다. 여러 조합을 비교해 보세요.

    자주 묻는 질문

    이 번호 생성기는 어떤 원리로 작동하나요?

    JavaScript의 난수 생성 함수를 사용하여 1부터 45까지의 숫자 중 중복 없이 6개를 무작위로 선택합니다. 매번 완전히 새로운 번호가 생성되며, 이전 결과와 무관합니다.

    생성된 번호로 당첨될 확률이 높아지나요?

    로또는 완전한 확률 게임이므로 어떤 번호 조합이든 당첨 확률은 동일합니다. 이 생성기는 편리하게 무작위 번호를 선택하는 도구이며, 당첨 확률을 높여주지는 않습니다.

    하루에 몇 번이나 사용할 수 있나요?

    횟수 제한 없이 무료로 무제한 사용 가능합니다.

    모바일에서도 사용할 수 있나요?

    네, 모든 기기(PC, 태블릿, 스마트폰)에서 사용할 수 있도록 반응형으로 설계되어 있습니다.
